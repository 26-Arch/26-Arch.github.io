课程信息

本课程通过一系列实验，实现一个 RISC-V 五级流水线 CPU 核，并使用 Verilator 仿真与 Difftest 进行验证。实验代码与说明通过 GitHub 仓库与 Wiki 发布。

推荐答疑方式

  • Issues：在项目仓库 26-Arch/26-Arch 发起
  • 微信群聊：信息量少时使用（不建议私聊，效率不高且容易忽略问题）
  • 线下：周二实验课

文档说明

课程材料分为两部分：代码仓库（26-Arch）和Wiki（说明文档）。请先克隆代码仓库并初始化子模块，再做实验。

如何获取与更新代码

在选定目录下执行：

git clone https://github.com/26-Arch/26-Arch.git
cd 26-Arch
git submodule update --init --recursive   # 初始化 difftest 子模块

每次助教发布新内容后，请先 commit 本地改动，再拉取并合并：

git fetch --all
git merge origin/main
git submodule update

代码需在 vsrc 目录下编写，核心在 vsrc/src/core.sv。不要修改 vsrc 以外的文件，除非你清楚自己在做什么。
